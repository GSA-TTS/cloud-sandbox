# AWS Service Features & Capabilities

Service-specific features extracted from documentation.

## Secrets Manager

**Source:** [AWS Service - Secrets Manager.md](AWS Service - Secrets Manager.md)

1. Programmatically Retrieve Encrypted Secret Values at Runtime Instead of Storing Them
2. Store Just About Any Kind of Secret
3. Automatically Rotate Any Controlled Secrets
4. Control Who Can Access Secrets
5. Compliant with HIPAA standards (possibly not business relevant)
